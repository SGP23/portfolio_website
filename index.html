About

Engineering Intelligent Systems

I'm an AI Engineer focused on building production-grade ML pipelines, vector retrieval systems, and interactive data applications. I specialize in turning unstructured data into searchable, queryable intelligence.

At Alchemyte Data Solutions, I engineered a hybrid semantic GIS assistant that combines LangChain RAG workflows with FAISS vector search over geospatial datasets — delivering spatial-aware Q&A through interactive Streamlit dashboards with Folium map overlays and Plotly analytics.

My work spans the full ML lifecycle: preprocessing automation, model training and compression, semantic embedding pipelines, vector indexing, and cloud deployment. I build systems that are designed for real users — with real data at production scale.
